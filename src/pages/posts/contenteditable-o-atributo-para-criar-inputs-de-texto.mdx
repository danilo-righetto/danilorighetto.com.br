Sabia que você consegue fazer qualquer elemento do HTML virar um input de texto? 🤯

O "contenteditable" é um atributo que permite tornar qualquer elemento HTML editável pelo usuário. Você pode usá-lo para criar editores de texto que você edita DIRETO na tag onde o elemento está, sem precisar trocar ou criar um novo input ou outros recursos interativos que você imaginar.

Para usar o "contenteditable", você precisa definir o atributo como "true" no elemento que deseja tornar editável, por exemplo:

```html
<p contenteditable="true">Este parágrafo é editável.</p>
```

Você também pode ouvir eventos de `input` no elemento editável para obter informações sobre as ações do usuário, como o objeto `dataTransfer` que contém os dados que foram colados ou soltos no elemento.

Por exemplo, você pode exibir os dados HTML que foram colados no elemento, assim:

```html
<p contenteditable="true">Cole algum conteúdo HTML aqui.</p>
<p id="result"></p>
<script>
  const editable = document.querySelector('[contenteditable="true"]');
  const result = document.getElementById('result');

  editable.addEventListener('input', (e) => {
    result.textContent = e.dataTransfer.getData('text/html');
  });
</script>
```
